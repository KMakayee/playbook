# Checkpoint

Write a state snapshot to `tasks/checkpoint.md` with:

1. **Current phase** — Research / Plan / Implement / Other
2. **What's done** — Completed steps (brief)
3. **What's next** — Immediate next step
4. **Key decisions made** — Choices that affect remaining work
5. **Open risks** — Anything unresolved

Keep it under 30 lines. Overwrite any existing checkpoint file.
Then confirm: "Checkpoint saved to tasks/checkpoint.md"
